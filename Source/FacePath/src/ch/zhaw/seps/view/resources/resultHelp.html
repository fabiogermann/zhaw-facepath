Ergebnis

Das Ergebnisfenster wird während der Suche fortlaufend mit den neusten Daten aktualisiert. Ist die Suche beendet, wird das Ergebnis farblich gekennzeichnet und die Knoten im Graphen werden mit den entsprechenden Profilnamen beschriftet.

Eine neue Suche können Sie durch einen Klick auf den Button "Neue Suche" starten. Nachdem Sie auf diesen Button geklickt haben, erscheint wieder die Suche.
