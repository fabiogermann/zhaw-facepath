Suche

Die Profile werden in der Form des Namens der Person, dessen Profil Sie angeben möchten, eingegeben. Existieren mehrere Profile mit dem gleichen Namen, so stellt Ihnen FacePath eine Auswahl mit den Profilbildern zur Verfügung, um das Profil eindeutig identifizieren zu können.

Sollten Sie trotz korrekter Angabe des Namens nicht das entsprechende Profil finden, konsultieren Sie die Fehleranalyse.

1.     Geben Sie unter "Start-Benutzernamen" den Namen des ersten Profils ein.

2.     Geben Sie unter "Ziel-Benutzernamen" den Namen des zweiten Profils ein.

3.     Nun findet die eindeutige Festlegung der zu verwendenden Facebook-Profile statt.
Klicken Sie auf "Benutzer suchen".

4.     FacePath sucht nach den Profilen, die den angegebenen Namen entsprechen und versucht das richtige Profil einzusetzen. Sollte das von FacePath ausgewählte Profil nicht stimmen, haben Sie die Möglichkeit, das richtige Profil auszuwählen. Sollten die Profile stimmen, können Sie zum Schritt 6 springen.
Klicken Sie auf den Pfeil neben dem Textfeld von "Start-Benutzernamen".
Es erscheint eine Liste der potenziell korrekten Benutzer, die dem eingegebenen Namen entsprechen.

5.     Klicken Sie in der Liste auf den gewünschten Benutzer. Wiederholen Sie diesen und den vorherigen Schritt für das zweite Profil, aber klicken Sie diesmal auf den zweiten Pfeil (neben dem Textfeld von "Ziel-Benutzernamen").
Sollte der Benutzer nicht in der Liste auffindbar sein, konsultieren Sie die Fehleranalyse.

6.     Nun haben Sie die Möglichkeit, die Suche zu konfigurieren.
Kreuzen Sie die gewünschten Optionen an. Sie können auch nichts ankreuzen.

7.     Nun haben Sie die gewünschten Profile ausgewählt, die Suchoptionen festgelegt und sind jetzt bereit, die Suche zu starten.
Klicken Sie auf den Button "Suchen" rechts unten in der Suche.

8.     Die Suche wurde nun gestartet.
