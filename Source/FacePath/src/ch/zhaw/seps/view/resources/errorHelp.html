Fehleranalyse

Login mit nicht möglich

1.     Vergewissern Sie sich, dass Sie Zugriff zum Internet haben.

2.     Rufen Sie die Seite www.facebook.com auf, um zu sehen, dass die Facebook-Server online sind. Falls es so ist, dass die Facebook-Server nicht erreichbar sind, müssen Sie warten bis diese wieder verfügbar sind.

3.     Diesen Schritt müssen Sie nur ausführen, wenn Sie sich mit Ihrem eigenen Facebook-Account einloggen möchten:
Loggen Sie sich mit Ihrem Facebook-Account auf der Webseite ein, um sicherzustellen, dass Ihr Login gültig ist. Ist dieser ungültig, überprüfen Sie nochmals Ihre Login-Daten und versuchen Sie das Passwort zurückzusetzen.

Sollte, trotz allen erfolgreich überprüften Schritten der Login nicht funktionieren, wenden Sie sich an die Entwickler von FacePath.

Gewünschte Person kann in der Suche nicht gefunden werden

1.     Stellen Sie nochmals sicher, dass Sie den Namen des Profils korrekt eingeben haben.

2.     Rufen Sie die Facebook-Webseite www.facebook.com auf und loggen Sie sich mit dem eigenen Facebook-Login ein.

3.     Suchen Sie nach dem Profil, das FacePath nicht finden konnte.
Sollten Sie das Profil auf Facebook nicht finden, dann ist das Profil nicht mehr verfügbar und kann auch nicht mehr für die Suche in FacePath verwendet werden.
Finden Sie das Profil auf Facebook, so fahren Sie bei Schritt 4 weiter.

4.     Öffnen Sie das Profil auf Facebook und sehen Sie Ihre Adressleiste in Ihrem Webbrowser an.
Die Leiste sieht in Etwa folgendermassen aus:

5.     Kopieren Sie den Text zwischen den Symbolen "/" und "?". Sollte kein Fragezeichen in der Adressleiste stehen, kopieren Sie den ganzen Text nach dem Symbol "/".


Was Sie gerade kopiert haben, ist die User-Identifikation eines Profils auf Facebook. Mittels dieser Identifikation sind Sie in der Lage das Profil eindeutig angeben zu können.

6.     Wechseln Sie zur Suche der FacePath-Applikation und fügen Sie die vorhin kopierte Identifikation in einer der beiden Textfelder ("Start-Benutzernamen" oder "Ziel-Benutzernamen").

7.     Das Profil wurde nun eindeutig angegeben und kann für die Suche in FacePath verwendet werden. Sie können den Webbrowser nun schliessen.
